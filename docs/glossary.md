<h1>Glossary</h1>

<h2>Decision</h2>
Probabilistic output indicating whether a subject meets a threshold (e.g. adult).

<h2>Threshold</h2>
Configured boundary (e.g. 18 years) used to classify decisions.

<h2>Score</h2>
Internal signal used to compute a decision. Not exposed directly.

<h2>Core</h2>
Service responsible for age signal extraction and decision logic.

<h2>AntiSpoof</h2>
Service responsible for detecting presentation attacks.

<h2>API</h2>
Gateway aggregating results into a unified response.

<h2>SDK</h2>
Client layer for interacting with the API.
